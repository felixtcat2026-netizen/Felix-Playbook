# TOOLS.md - Local Notes

This file is for your environment-specific details. Things your agent needs to know about YOUR setup.

## What Goes Here

- Camera names and locations
- SSH hosts and aliases
- Preferred voices for TTS
- Device nicknames
- Anything unique to your setup

## Examples

```
### SSH
- home-server -> 192.168.1.100, user: admin

### TTS
- Preferred voice: "alloy"
- Default speaker: Kitchen HomePod

### Smart Home
- Living room lights: "main-lights" in HomeKit
- Front door camera: "front-cam"
```

Skills define HOW tools work. This file is for YOUR specifics. Keeping them separate means you can update skills without losing your notes.
